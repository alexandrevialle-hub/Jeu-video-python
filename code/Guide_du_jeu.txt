Guide du jeu

Lorsque vous lancez le jeu avec F5, appuyez sur M pour désactiver la barre des améliorations (vous pouvez la faire réapparaitre en appuyant sur M).
Vous pouvez vous déplacer dans le jeu et dans la barre des améliorations avec les flèches directionnelles.
La touche espace permet d'attaquer et dans la barre des améliorations, elle permet de choisir d'améliorer une des compétences proposés.
Cette amélioration coûte de l'exp qu'on récupère en tuant les mobs dans le jeu.
Pour utiliser la magie, appuyez sur ctrl (attention il y a une barre de mana).
La touche e permet de changer la magie et la touche q permet de changer l'arme.
